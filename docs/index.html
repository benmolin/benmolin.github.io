About

SNAP Screener

The mission of this website and SNAP prescreener is to help connect individuals to SNAP benefits and to make SNAP eligibility data more accessible.

This project is open source, meaning that anybody can see the code used to power it, collaborate on the project, and re-use the project however they'd like.

The original code for this project was written by 18F, a government organization. You can read more about 18F's history on the project here.

This project has no affiliation with 18F, instead building on top of the code they started. Please see the disclaimer at the bottom of this page.

This website is run by a group of volunteers to create SNAP technology resources for individuals as well as nonprofit organizations (such as food banks) to use.

We are working to compile all the miscellaneous resources about SNAP eligibility from the USDA, state handbooks, and federal government into one easily accessible source.

If you are interested in contributing to this project, we would welcome assistance on compiling additional helpful materials and reaching out to food banks.

If you have any feedback or questions, please send us an email.

Disclaimer

All estimates and data on this website are for educational purposes only and provide a basic estimate based on publicly available information. State test, benefit, and allowance amounts may be incomplete, incorrect, or outdated due to limitations in finding updated sources. Eligibility factors vary between states, and some deductions and eligibility criteria will not be included here. The only way to find out your true eligibility and benefit amount is to apply.
